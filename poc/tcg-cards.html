Card Anatomy

Five zones. One glance.

Every TCG card solves the same problem: communicate type, status, and value in a fixed frame. So does every deliverable.

1
Identity Zone
Deliverable ID (monospace), name (2-line max), and energy pips showing effort cost at a glance.
2
Type Bar
Full-width structural divider. Type icon + FEATURE/BUGFIX/REFACTOR + current SDLC status. Replaces the typical badge stack.
3
Ability Text
What the deliverable does (primary) and why it matters (italic flavor text). Two-tier hierarchy from every TCG ever made.
4
Artifact Pills
Doc-type colored: SPEC (blue), PLAN (teal), RESULT (green). Shows which SDLC artifacts exist for this deliverable.
5
Stat Block
IMP/EFF numbers (like ATK/DEF) plus a set symbol showing the sprint or milestone this card belongs to.
