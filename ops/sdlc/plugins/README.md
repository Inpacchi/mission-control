# Plugin Dependencies

## Optional

| Plugin | What It Does | Skills That Use It |
|--------|-------------|-------------------|
| **oberskills** | Prompt engineering (`oberprompt`), web research (`oberweb`), skill creation (`obercreate`), and prose editing (`oberscribe`). | Available to all skills but not required by any |
| **design-for-ai** | Enriches design consultations with design theory from *Design for Hackers*. | `design-consult` |

Both plugins are quality enhancements — SDLC skills function without them but produce better output with them.

See `oberskills-setup.md` and `design-for-ai-setup.md` for installation instructions.
